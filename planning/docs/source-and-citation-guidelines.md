# Source and Citation Guidelines

## Core Standards

The project requires traceable, cited factual claims and original wording. Every report and final post should include a `## References` section.

## Source Handling

- Cite the source URL, source file, paper record, or repository record that supports each factual claim.
- Use quotation marks for any exact quoted text.
- Do not publish raw scrape output as article prose.
- Distinguish confirmed facts from commentary, interpretation, or uncertainty.

## Review Checklist

- [ ] Every factual claim can be traced to a source.
- [ ] Sources are listed in `## References`.
- [ ] Any direct quotation is clearly quoted and cited.
- [ ] Unsupported claims are removed or rewritten.
- [ ] Raw collection output is not committed or copied into public content.
