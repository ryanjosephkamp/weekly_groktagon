# Publication Checklist

Use this checklist before publishing each weekly final post.

## Content Completeness

- [ ] The week folder is named `weekly/YYYY-WXX/`.
- [ ] The final post includes a centered `/assets/groktagon-logo.png` logo.
- [ ] The final post includes exactly eight main report sections.
- [ ] The final post includes a clickable table of contents.
- [ ] The final post includes a conclusion and links section.
- [ ] The final post includes `## References`.

## Citation and Wording

- [ ] Every factual claim is source-traceable.
- [ ] Direct quotations are quoted and cited.
- [ ] Wording is original.
- [ ] Unsupported claims have been removed.

## Public/Private Separation

- [ ] No raw scrape output is committed.
- [ ] No scrape manifest or sanitized scrape summary is committed.
- [ ] No `weekly/*/raw/` content is staged.
- [ ] First workflow artifact has been manually reviewed before relying on automation for the pilot.
