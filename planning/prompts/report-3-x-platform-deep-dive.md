# Report 3 Prompt — X Platform Deep Dive

## Scope

Create Report 3 from X-platform material for the target week, using strict relevance filtering.

## Requirements

- Include only material that is clearly relevant to the weekly topic.
- Distinguish official, high-signal, and speculative posts.
- Use original wording.
- Cite every factual claim with a source URL.
- Do not treat unsourced social posts as confirmed facts.
- End with a `## References` section.

## Output Structure

1. Title
2. Short overview
3. High-signal posts and themes
4. Claims that require caution
5. Excluded or low-confidence categories
6. `## References`
