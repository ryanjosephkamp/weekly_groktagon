# Report 1 Prompt — Official xAI Sources Summary

## Scope

Create Report 1 from the collected official xAI source files for the target week. Use only the provided weekly source material for this report.

## Requirements

- Summarize only material that is relevant to the target week.
- Use original wording.
- Cite every factual claim with the source file or source URL available in the weekly material.
- Do not include unsupported claims.
- Do not publish raw scrape text wholesale.
- End with a `## References` section.

## Output Structure

1. Title
2. Short overview
3. Key developments
4. Notable updates or changes
5. Open questions or gaps, if any
6. `## References`
