# Report 7 Prompt — Practical Tutorials & Vibe-Coding Spotlight

## Scope

Create Report 7 by curating 1 to 3 reproducible practical examples or workflows from the target week.

## Requirements

- Include only examples that are reproducible from cited source material.
- Use original wording.
- Cite every factual claim with a source URL.
- Clearly identify prerequisites and limitations.
- Avoid presenting untested workflows as guaranteed outcomes.
- End with a `## References` section.

## Output Structure

1. Title
2. Short overview
3. Spotlight example 1
4. Spotlight example 2, if applicable
5. Spotlight example 3, if applicable
6. Reproducibility notes
7. `## References`
