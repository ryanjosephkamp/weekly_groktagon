# Report 5 Prompt — Community & Ecosystem Insights

## Scope

Create Report 5 from community and ecosystem sources outside X, such as developer forums, code repositories, discussion boards, or technical communities.

## Requirements

- Focus on adoption signals, practical uses, integrations, tips, and sentiment.
- Use original wording.
- Cite every factual claim with a source URL.
- Distinguish individual anecdotes from broader patterns.
- Do not include unsupported claims.
- End with a `## References` section.

## Output Structure

1. Title
2. Short overview
3. Adoption and usage signals
4. Practical community examples
5. Sentiment and limitations
6. `## References`
